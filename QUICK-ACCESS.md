## ❓ FAQ

**Q: How do I filter multiple selections?**
A: Each filter applies independently. Select Region, then Category, and both filters work together.

**Q: Why is Laptop an outlier?**
A: It's a high performer, not a data error. It legitimately generates the most revenue.

**Q: What does "G rowth Rate" mean?**
A: It's the percentage increase from one quarter to the next. Example: 15.8% means Q2 sales are 15.8% higher than Q1.

**Q: Can I export the data?**
A: Currently, you can use browser's built-in tools. Future versions will have export buttons.

**Q: What if I want to upload my own data?**
A: Click the main "Upload File" button and select your CSV file. The dashboard will auto-generate.

**Q: How accurate are the calculations?**

## 📞 Support

If you encounter any issues:

1. **Dashboard not loading?**
   - Check if port 5174 is accessible
   - Refresh browser (Ctrl+F5)
   - Check browser console for errors (F12)

2. **Filters not working?**
   - Check browser console for JavaScript errors
   - Try different filter combinations
   - Refresh the page

3. **Charts not displaying?**
   - Ensure JavaScript is enabled
   - Try different browser (Chrome, Firefox, Edge)
   - Clear browser cache

4. **Want to analyze different data?**
   - Click "← Back to Upload"
   - Select a different CSV file
   - Dashboard will regenerate
