# 📊 AI-Powered Dashboard Generator

An intelligent dashboard generator that uses Google Gemini AI to automatically create beautiful, interactive dashboards from your data files.

## 🌟 Features

- **📁 Multi-format Support**: Upload CSV, Excel, or JSON files
- **🤖 AI-Powered Analysis**: Google Gemini AI automatically analyzes your data and suggests relevant visualizations
- **📈 Interactive Charts**: Beautiful charts with Chart.js (Bar, Line, Pie, Scatter)
- **💡 Key Metrics**: Automatically calculated KPIs and summary statistics
- **🔍 Data Tables**: Sortable and filterable data tables
- **✨ Natural Language Design**: Describe your dashboard with natural language for custom layouts
- **🎨 Responsive Design**: Works seamlessly on desktop and mobile

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Google Gemini API Key

### Installation

#### 1. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file with your Gemini API key:

```
MONGODB_URI=mongodb+srv://your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_google_gemini_api_key
```

**Get Gemini API Key:**
1. Visit [Google AI Studio](https://ai.google.dev/tutorials/setup)
2. Click "Get API Key"
3. Create a new API key and copy it to your `.env` file

#### 2. Frontend Setup

```bash
cd frontend
npm install
```

### Running the Application

#### Start Backend Server

```bash
cd backend
npm run dev
```

The backend will run on `http://localhost:3000`

#### Start Frontend Development Server

In a new terminal:

```bash
cd frontend
npm run dev
```

The frontend will typically run on `http://localhost:5173`

## 📝 How to Use

### Step 1: Upload Data
1. Open the application in your browser
2. Click on the upload area to select a file (CSV, Excel, or JSON)
3. Optionally add a description of what you want in your dashboard
4. Click "Generate Dashboard"

### Step 2: Review Dashboard
Once your data is processed:
- **Key Metrics** - Summary statistics are automatically calculated
- **AI Insights** - Natural language summary of your data
- **Visualizations** - Charts automatically generated by AI
- **Data Table** - Full dataset with sorting and filtering

### Step 3: Customize (Optional)
- Use the "Customize Dashboard" section to regenerate with specific requirements
- Click on any chart to expand and view it in full size

### Example Data
A sample `sample_data.csv` is included. Test with your own data or use this example:
- Product sales data across quarters and regions
- Multiple categories (Electronics, Accessories, Furniture)
- Perfect for demonstrating sales trend analysis

## 🔌 API Endpoints

### Dashboard Routes

#### POST `/api/dashboard/upload`
Upload a file and generate dashboard

**Request:**
```
multipart/form-data
- file: File (CSV, Excel, or JSON)
- prompt: (optional) Natural language description
```

**Response:**
```json
{
  "success": true,
  "sessionId": "string",
  "fileInfo": {
    "name": "string",
    "type": "string",
    "rowCount": number,
    "columnCount": number
  },
  "columns": ["string"],
  "dashboardConfig": {
    "dashboardTitle": "string",
    "insights": ["string"],
    "charts": [
      {
        "id": "string",
        "title": "string",
        "type": "bar|line|pie|scatter|table",
        "xAxis": "string",
        "yAxis": "string|string[]",
        "description": "string"
      }
    ],
    "metrics": [
      {
        "id": "string",
        "label": "string",
        "column": "string",
        "aggregation": "sum|avg|count|max|min"
      }
    ]
  },
  "insights": "string",
  "data": [{}],
  "totalRows": number
}
```

#### POST `/api/dashboard/regenerate`
Regenerate dashboard with custom prompt

**Request:**
```json
{
  "data": [{}],
  "columns": ["string"],
  "prompt": "string"
}
```

#### POST `/api/dashboard/calculate-metrics`
Calculate aggregated metrics

**Request:**
```json
{
  "data": [{}],
  "metrics": [
    {
      "id": "string",
      "column": "string",
      "aggregation": "sum|avg|count|max|min",
      "label": "string"
    }
  ]
}
```

## ⚠️ Important Notes

### API Key Security
- **NEVER** commit your Gemini API key to version control
- Always use environment variables
- Use `.env` file for local development (included in `.gitignore`)

### File Size Limits
- Default max file size: 50MB
- Adjust in backend `server.js` if needed
- Supported formats: CSV, XLSX, XLS, JSON

### Data Privacy
- Data is processed temporarily during dashboard generation
- Files are deleted after processing
- No data is stored in the system by default

## 🚨 Troubleshooting

### Backend won't start
```bash
# Check Node version
node --version

# Clear node_modules and reinstall
rm -rf node_modules
npm install
```

### API connection errors
- Ensure backend is running on port 3000
- Check CORS is enabled in `server.js`
- Verify firewall settings

### Gemini API errors
- Verify API key is valid and active
- Check if you have API quota remaining
- Ensure API is enabled in Google Cloud Console

### File upload fails
- Check file format (CSV, XLSX, XLS, JSON supported)
- Verify file size is under 50MB
- Ensure file contains valid data
